# Text Typer 2.0

![Text Typer Logo](logo-readme.png)

## Описание

Text Typer — это приложение для автоматической печати текста с поддержкой гибких настроек. Программа имитирует нажатия клавиш, позволяя автоматически вводить текст в любое приложение.

[![Build Status](https://github.com/king-tri-ton/TextTyper/workflows/Build%20and%20Scan%20Text%20Typer/badge.svg)](https://github.com/king-tri-ton/TextTyper/actions)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Python Version](https://img.shields.io/badge/python-3.12-blue.svg)](https://www.python.org/)

## Возможности

- **Современный интерфейс** на базе PySide6
- **Глобальные горячие клавиши** — запуск из любого приложения
- **Гибкие настройки**:
  - Настраиваемая задержка перед началом (1-30 сек)
  - Регулируемая скорость печати (в миллисекундах)
  - Выбор горячей клавиши (F9, F10, F11, F12, Ctrl+Shift+T, Ctrl+Alt+T)
- **Контроль выполнения** — остановка печати в любой момент

## Установка

### Из исходников

1. Клонируйте репозиторий:
```bash
git clone https://github.com/king-tri-ton/TextTyper.git
cd TextTyper
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python app.py
```

### Скачать готовый EXE

Скачайте последнюю версию из [раздела Releases](https://github.com/king-tri-ton/TextTyper/releases)

## Использование

1. **Вставьте текст** в текстовое поле
2. **Настройте параметры**:
   - Задержка перед началом
   - Скорость печати
   - Горячая клавиша
3. **Запустите печать**:
   - Нажмите кнопку "Начать печатать"
   - Или используйте назначенную горячую клавишу
4. **Переключитесь** на нужное окно в течение заданной задержки
5. При необходимости используйте кнопку **"Остановить"**

## Сценарии использования

- Автоматический ввод длинных текстов
- Обход ограничений на копирование/вставку
- Демонстрация "живой" печати
- Тестирование форм ввода
- Заполнение повторяющихся данных

## Требования

- **ОС**: Windows 10/11
